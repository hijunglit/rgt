# rgt 사전 과제

### 서비스 구동 방법

1. 레포지토리 클론
2. 터미널을 사용해 클론한 디렉토리 위치에서 cd accessingdatamongodb
3. ./gradlew bootRun 명령어로 spring boot 구동
4. 클론한 디렉토리 위치에서 cd client
5. npm start 명령어로 react 구동
